# Entrevista — Inversionista experimentado

> Notas crudas de la conversación. Entrevista simulada y anonimizada para identificar necesidades de usuarios con experiencia en análisis financiero y trading.

**Entrevistador:** ¿Cómo toma actualmente sus decisiones de inversión?

**T. (inversionista experimentado):** Normalmente reviso gráficos, indicadores técnicos, noticias del mercado y comportamiento histórico. No me basta con que una aplicación diga “compra” o “vende”; necesito ver el fundamento.

**Entrevistador:** ¿Cuál es el dolor más grande en las herramientas actuales?

**T.:** Muchas plataformas tienen datos, pero no integran bien el análisis. Uno tiene que estar saltando entre varias herramientas para ver indicadores, comparar activos y configurar alertas.

**Entrevistador:** ¿Qué esperaría de una aplicación de este tipo?

**T.:** Datos actualizados, indicadores técnicos configurables y alertas personalizadas. También me interesa poder comparar activos y guardar estrategias.

**Entrevistador:** ¿Qué indicadores considera importantes?

**T.:** RSI, MACD, medias móviles, volatilidad, volumen y comportamiento histórico. También sería útil ver señales combinadas, no indicadores aislados.

**Entrevistador:** ¿Qué sería una mala experiencia para usted?

**T.:** Datos retrasados, recomendaciones genéricas o que el sistema no explique qué variables utilizó para generar una señal.
